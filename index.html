PROJETOS PROFISSIONAIS

Anatomap

Anatomap é um game de exploração do corpo humano onde você aprende sobre os sistemas, músculos e esqueleto. O projeto utiliza tecnologias como Dotween para animações, sistema utilizando Cinemachine para zoom nas partes do corpo selecionadas pelo usuário, Sistema para controlar a opacidade e transparência dos materiais para dar o efeito de "highlight" nas partes selecionadas, utilização de Shaders para highlight nas partes interativas onde o usuário passar o mouse, além de um sistema completo de locuções para todo o aplicativo, tornando-o acessível.

Modulab

Modulab é um conjunto multidisciplinar de laboratórios e atividades em realidade aumentada, abrangendo conteúdos de Matemática, Biologia, Astronomia, Física e outras áreas do conhecimento. O projeto utiliza tecnologias como Dotween para animações, Scriptable Objects para gerenciamento de dados, ARFoundation para a realidade aumentada, sistema de interação com modelos 3D em realidade aumentada (rotação, zoom e centralização de objetos na tela do dispositivo), além de um sistema de locução contemplando todo o projeto, para acessibilidade.

Caminho das Reduções

Jogo educacional com a temática das reduções jesuíticas, onde os jogadores avançam em um tabuleiro interativo, participam de minijogos como quebra-cabeça e caça-palavras, acumulando pontos e competindo entre si. O projeto utiliza tecnologias como Dotween para animações, Scriptable Objects para gerenciamento de dados, o Event System da Unity e um sistema de grid personalizado.


EducaPuzzle

Jogo educacional inspirado em "Fall Guys" onde o jogador deve responder perguntas pré-determinadas para prosseguir no jogo, unindo elementos de jogos casuais com elementos educacionais. O projeto utiliza tecnologias como Dotween, State Machines, Event System, Scriptable Objects, além de integrações com APIs de login e pontuação para autenticação e classificação dos jogadores.

Despertar Socioemocional

Experiência inovadora em realidade aumentada voltada para a educação socioemocional de crianças e adolescentes. Esse projeto de RA oferece uma experiência interativa e enriquecedora, onde o usuário pode explorar suas emoções, identificar áreas de interesse e desenvolver autoconhecimento. Utilizando tecnologias de ponta, como filtros faciais e IA para reconhecimento de emoções, o jogo permite que o usuário interaja de forma intuitiva através do movimento da cabeça e reconhecimento facial. Essa abordagem lúdica e imersiva busca fomentar o desenvolvimento emocional de forma envolvente e educativa. O projeto utiliza tecnologias como Dotween para animações, Scriptable Objects para gerenciamento de dados, ARFoundation para a realidade aumentada, utilização de inteligência artificial para reconhecimento de emoções simples, além de detecção facial e aplicação de filtros no usuário.

Coleção Entrelaços

Experiência em realidade aumentada, elaborada para a educação infantil, trazendo vida a diferentes animais, assim como mostrando diversos alimentos em diferentes apresentações. A experiência auxilia no processo de fala e alfabetização de crianças, utilizando a realidade aumentada como um importante aliado. O projeto utiliza tecnologias como Dotween para animações, Scriptable Objects para gerenciamento de dados, ARFoundation para a realidade aumentada, interações simples com os objetos da tela, como rotação e zoom, além de um sistema completo de audiodescrição e sons dos animais presentes na experiência.

Match 3

Protótipo de jogo no estilo "Match 3", utilizando um sistema de Grid dinâmico para gerenciar as peças e suas combinações, eventos e sistemas de feedback para animações e efeitos visuais responsivos, Dotween, para animações dos matches e também da seleção e desseleção das peças, Object Pooling para os efeitos visuais de explosão de gemas, garantindo a máxima performance.

Tio Patinhas Coletor

O jogo utiliza uma arquitetura orientada a eventos onde moedas são geradas periodicamente, e o Tio Patinhas responde a esses eventos para atualizar seu alvo. Além disso, o jogo utiliza uma implementação personalizada de uma lista ligada ordenada para gerenciar as moedas. Cada moeda é adicionada à lista em ordem decrescente com base na sua distância ao Tio Patinhas. Isso garante que ele sempre direcione para a moeda mais próxima, otimizando o caminho de coleta.

Professor Preguiçoso

Jogo de travessia de grafo, implementando o Algoritmo de Dijkstra para descobrir o menor caminho do ponto inicial ao ponto final. Projeto criado para exercitar conceitos de "pathfinding" com listas linkadas ordenadas (sorted linked lists).

WebGL


Quick Shooter
