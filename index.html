PROJETOS PROFISSIONAIS

Caminho das Reduções

Jogo educacional com a temática das reduções jesuíticas, onde os jogadores avançam em um tabuleiro interativo, participam de minijogos como quebra-cabeça e caça-palavras, acumulando pontos e competindo entre si. O projeto utiliza tecnologias como Dotween para animações, Scriptable Objects para gerenciamento de dados, o Event System da Unity e um sistema de grid personalizado.


EducaPuzzle

Jogo educacional inspirado em "Fall Guys" onde o jogador deve responder perguntas pré-determinadas para prosseguir no jogo, unindo elementos de jogos casuais com elementos educacionais. O projeto utiliza tecnologias como Dotween, State Machines, Event System, Scriptable Objects, além de integrações com APIs de login e pontuação para autenticação e classificação dos jogadores.

PROJETOS PESSOAIS

Tio Patinhas Coletor

O jogo utiliza uma arquitetura orientada a eventos onde moedas são geradas periodicamente, e o Tio Patinhas responde a esses eventos para atualizar seu alvo. Além disso, o jogo utiliza uma implementação personalizada de uma lista ligada ordenada para gerenciar as moedas. Cada moeda é adicionada à lista em ordem decrescente com base na sua distância ao Tio Patinhas. Isso garante que ele sempre direcione para a moeda mais próxima, otimizando o caminho de coleta.

Professor Preguiçoso

Jogo de travessia de grafo, implementando o Algoritmo de Dijkstra para descobrir o menor caminho do ponto inicial ao ponto final. Projeto criado para exercitar conceitos de "pathfinding" com listas linkadas ordenadas (sorted linked lists).
