PROFESSIONAL PROJECTS

Anatomap

Anatomap is an exploration game of the human body where players learn about its systems, muscles, and skeleton. The project incorporates technologies such as DOTween for animations, a Cinemachine-based system for zooming in on body parts selected by the user, and a material opacity and transparency control system to create a highlight effect on selected parts. It also uses shaders to highlight interactive areas when hovered over with the mouse. Additionally, the game features a comprehensive voice-over system, making it fully accessible.

Modulab

Modulab is a multidisciplinary set of laboratories and activities in augmented reality, covering subjects such as Mathematics, Biology, Astronomy, Physics, and other areas of knowledge. The project utilizes technologies such as DOTween for animations, Scriptable Objects for data management, ARFoundation for augmented reality, a system for interaction with 3D models in augmented reality (rotation, zoom, and object centering on the device screen), and a voice-over system covering the entire project for accessibility.

Caminho das Reduções

An educational game with the theme of Jesuit reductions, where players advance on an interactive board, participate in minigames such as puzzles and word searches, accumulating points and competing against each other. The project uses technologies such as DOTween for animations, Scriptable Objects for data management, Unity's Event System, and a custom grid system.


EducaPuzzle

An educational game inspired by 'Fall Guys' where the player must answer predetermined questions to progress in the game, combining elements of casual gaming with educational elements. The project utilizes technologies such as DOTween, State Machines, Event System, and Scriptable Objects, along with integrations with login and scoring APIs for player authentication and ranking.

Socioemotional Awakening (Despertar Socioemocional)

An innovative augmented reality experience aimed at promoting socioemotional education for children and adolescents. This AR project offers an interactive and enriching experience where users can explore their emotions, identify areas of interest, and develop self-awareness. Using cutting-edge technologies such as facial filters and AI for emotion recognition, the game allows users to interact intuitively through head movements and facial recognition. This playful and immersive approach seeks to foster emotional development in an engaging and educational way. The project utilizes technologies like Dotween for animations, Scriptable Objects for data management, ARFoundation for augmented reality, AI for basic emotion recognition, as well as facial detection and filter application for the user.

Entrelaces Collection (Coleção Entrelaços)

An augmented reality experience designed for early childhood education, bringing various animals to life and showcasing different foods in diverse presentations. This experience supports children's speech and literacy development, using augmented reality as a valuable ally. The project utilizes technologies like Dotween for animations, Scriptable Objects for data management, ARFoundation for augmented reality, simple interactions with screen objects, such as rotation and zoom, along with a comprehensive audio description system and animal sounds featured in the experience.

Uncle Scrooge Collector

The game uses an event-driven architecture where coins are generated periodically, and Uncle Scrooge responds to these events to update his target. Additionally, the game features a custom implementation of a sorted linked list to manage the coins. Each coin is added to the list in descending order based on its distance from Uncle Scrooge. This ensures that he always targets the nearest coin, optimizing the collection path.

WebGL

Lazy Professor

Graph traversal game implementing the Dijkstra's Algorithm to find the shortest path from the starting point to the endpoint. This project was created to practice pathfinding concepts using sorted linked lists.
